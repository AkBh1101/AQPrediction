# Air Quality Detection & Hazard Prediction (ML Project)

## Overview
This project focuses on analyzing air quality data and predicting whether conditions are hazardous using Machine Learning techniques. It includes data preprocessing, exploratory data analysis (EDA), model building, and performance evaluation.

---

## Objectives
- Analyze air quality data to identify patterns and trends  
- Perform data preprocessing and feature selection  
- Build and compare multiple ML models  
- Predict hazardous conditions based on input features  

---

## Tech Stack
- **Programming Language:** Python  
- **Libraries:** NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn  
- **Tools:** Jupyter Notebook / VS Code  

---

## Machine Learning Models Used
- Logistic Regression  
- Decision Tree Classifier  
- K-Nearest Neighbors (KNN)  

---

## Workflow
1. Data Loading  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Train-Test Split  
6. Model Training  
7. Model Evaluation (Accuracy, Confusion Matrix)  

---

## Results
- Compared performance of multiple models  
- Identified optimal parameters (e.g., best K for KNN using elbow method)  
- Evaluated model accuracy and classification performance
